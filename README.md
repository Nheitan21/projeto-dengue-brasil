# 📊 Análise da Dengue no Brasil

## 📌 Problema
A dengue é uma das principais doenças transmitidas por mosquitos no Brasil. Este projeto tem como objetivo analisar a distribuição dos casos ao longo do tempo e entre os estados brasileiros.

---

## 🎯 Objetivo
Identificar padrões de ocorrência da dengue no Brasil e entender quais regiões são mais afetadas.

---

## 📁 Fonte dos Dados
Dataset público contendo registros de casos de dengue por estado no Brasil ao longo do tempo.

---

## 🛠️ Ferramentas Utilizadas
- Python (Pandas)
- Google Colab
- Looker Studio

---

## 🔄 Processamento dos Dados
- Transformação do dataset para formato tabular (melt)
- Conversão de datas
- Agregações por estado e ano

---

## 📊 Análises Realizadas
- Casos por estado
- Evolução temporal dos casos
- Distribuição dos casos entre estados

---

## 📌 Insights
- Os casos de dengue variam significativamente entre os estados brasileiros
- Estados como São Paulo, Minas Gerais e Bahia concentram maior número de casos
- Existe comportamento sazonal ao longo do tempo
- A distribuição dos casos não é uniforme entre regiões
